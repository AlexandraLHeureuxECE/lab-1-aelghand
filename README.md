Features
- 3×3 grid, alternating turns, win/draw detection, restart anytime
- Required extra feature implemented:
Keyboard support:
  - Arrow keys move focus
  - Enter/Space places a mark
  - R restarts game
- Extra polish:
  - Winning line highlighted
  - Theme toggle (light/dark)
  - Disabled cells after being played
